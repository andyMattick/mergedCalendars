    Planner Help
    Quick guidance for first-time setup and daily use.

    Welcome to Wesleyan HS Planner

    This tool helps you plan your classes with rotation days, units, notes, and test days — all saved locally in your browser.

    • Enter your class names on the left
    • Define units per period (with topics, quizzes, tests)
    • Click any day to add notes or mark test days
    • Use Backup/Restore to save your work
    • Switch between semesters using the selectors in the header

    Everything works offline.

    Units

    Units drive automatic pacing on the calendar.

    How to use:

    • Enter Unit Name and Length → + Add Unit
    • Click "Edit Days" to set topics and mark quizzes/tests per day
    • Calendar will show: AP Stats - Unit 1: Exploring Data (Day 4/10)

    Snow Day & Unit Shifting

    If you have a snow day or need to adjust pacing:

    • Go to the unit in the left panel and click "+1 Day (Snow)" or "-1 Day".
    • This shifts the selected unit and all following units.
    • You can also shift from the day modal.
    • The app warns you if any tests would land on a No Test day after shifting.

    Special Days

    Mark Family Nights, holidays, PSAT, etc. You can add multiple special tags per day and remove them individually.

    Days after major holidays are automatically No Test (you can override).

    Notes & Assessments

    Click any day on the calendar to open the day modal.

    • Per-period notes
    • Test Today checkbox per class period (creates a "Test" badge on that class)
    • NO TEST TODAY toggle (separate from special tags — blocks tests on the whole day)
    • Quizzes/Tests planned in the Unit Day Planner appear automatically

    Conflict warnings appear if you try to schedule a test on a No Test day (or vice versa).

    Printing & Export

    • Print with Filters — clean monthly view
    • Export Tests/Quizzes by Class — new button in header (copies clean list to clipboard)
    • Always use Backup/Restore to save your data safely.
